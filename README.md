# Django Real Estate

A real estate listing site made with Django 

## Installation

Clone the repo

```bash
https://github.com/teamallnighter/btre_project.git
```

## Features
 - Dynamic rendering of the listings
 - Login for Django back to end to manage the listings and realtors
 - MVP Realtors
 - Realtor Bios
